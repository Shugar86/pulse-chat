> Глобальный канон для агентов: `~/dev/agent-os/AGENTS.md` и `~/dev/agent-os/INDEX.md`.

## Базовые принципы

- **Язык:** отвечаем на русском, если пользователь не попросил иначе.
- **KISS:** минимальные изменения, самое простое решение.
- **Тестируй:** после изменений запускаем проверки/тесты.
- **Не ломай:** если не уверен — спрашиваем, не додумываем.
- **Git:** не коммитим и не пушим без явного разрешения пользователя.

## Стек

- Expo / React Native (TypeScript)
- Node.js + Express + Socket.io
- PostgreSQL + Prisma
- Docker Compose, pnpm workspaces
